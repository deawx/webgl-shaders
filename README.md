WebGL Fractal Renderer
--------
![main image](http://jonathan-potter.github.io/webgl-shaders/assets/julia-set.png "Julia Set")

Instructions:
--------

run using:
```
npm start
```
open this url in your browser.
```
http://localhost:8080/
```
